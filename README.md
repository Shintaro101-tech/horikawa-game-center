# 堀川家のゲームセンター 🎮

家族で遊べるブラウザゲーム集。スマホ・タブレット・PC・ゲームパッドに対応。

## 🕹 ゲーム一覧

### 🛥 [深海大冒険](deep-sea/)
潜水艦・バギー・ジェット・宇宙船で 4ワールド × 5ステージを冒険するシューティング。
コインを集めてレベルアップ、新しい乗り物をアンロック！

- ライフ → HP 数値表示、ダメージポップアップ
- 5種類の特殊武器（レーザー / 貫通ミサイル / なかま召喚 など）
- ボス戦は専用 BGM + 派手な演出

### 🏍 [Kanjiレースバトル](kanji-race/)
近未来 3D レースで漢字の読みに合うコース（ドア）を選ぶ学習ゲーム。
- 1年生のかんじ（80字）
- 2年生のかんじ（学習日に合わせて随時追加）

10問正解でクリア。間違えるとドアに激突して粉々にクラッシュ！

## 🎮 操作方法

| 操作 | 入力 |
|---|---|
| 移動 | タッチ・マウス・矢印キー・ゲームパッド左スティック / 十字キー |
| 決定 | タップ・Enter / Space・ゲームパッド A / Start |
| 戻る | Esc・ゲームパッド B / Back |

## 📱 対応環境

- iOS Safari / Android Chrome
- iPad / Android タブレット
- Mac / Windows ブラウザ
- Bluetooth / USB ゲームパッド

## 🌐 公開・共有

GitHub Pages 推奨。リポジトリの **Settings → Pages → Deploy from a branch (main / root)** で有効化すると：

```
https://YOUR_NAME.github.io/REPO_NAME/
```

でアクセス可能になります。

## 🛠 構成

```
.
├── index.html        ポータル（ゲーム選択画面）
├── deep-sea/         深海大冒険
└── kanji-race/       Kanjiレースバトル
```

各ゲームは **Phaser 3** (CDN 経由) を使用。サーバー不要で静的ホスティングのみで動作します。
